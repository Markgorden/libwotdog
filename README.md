# 号称百万级TCP在线服务器框架

1、占用内存少，事件模型，高效，百万级TCP在线。主要用于百万级物联网设备的入网连接。
2、同一端口复用各种通用协议。
3、默认实现http协议。
4、默认实现ws协议。
5、随意添加私有二进制协议。
